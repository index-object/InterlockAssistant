# 剪贴板监视助手 - 设计文档

## 1. 项目概述

Windows系统剪贴板监视工具，可显示剪贴板内容来源窗口，支持过滤、检索和快捷键操作。

## 2. 技术栈

- Python 3.8+
- PySide2 (GUI框架)
- pyclipboard (剪贴板监控)
- win32gui (窗口信息获取)
- QHotkey (全局快捷键)
- SQLite (本地数据库)

## 4. 功能模块

### 4.1 剪贴板监控 (ClipboardWatcher)
- 实时监控剪贴板变化
- 仅处理文本内容
- 过滤重复内容

### 4.2 窗口信息获取 (WindowInfo)
- 获取当前活动窗口句柄(hwnd)
- 获取窗口标题 (win32gui.GetWindowText)
- 支持过滤指定窗口

### 4.3 快捷键管理 (HotkeyManager)
- 全局热键显示/隐藏悬浮窗
- 用户可自定义快捷键
- 保存到配置文件

### 4.4 数据库检索 (DatabaseService)
- SQLite存储关键词
- 剪贴板文本模糊匹配
- 返回匹配结果

### 4.5 系统托盘 (TrayManager)
- 最小化到托盘
- 右键菜单: 显示/隐藏、退出
- 双击托盘图标显示窗口

### 4.6 窗口过滤配置 (FilterConfig)
- 枚举当前运行窗口
- 勾选要监控的软件
- 保存到JSON配置

## 5. 界面设计

### 5.1 悬浮窗 (置顶)

```
┌────────────────────────────────┐
│  [📋] Notepad++ - file.txt  × │
├────────────────────────────────┤
│  剪贴板内容预览:               │
│  这是剪贴板中的文字内容...     │
├────────────────────────────────┤
│  检索结果:                     │
│  • 关键词A → 相关内容         │
│  • 关键词B → 相关内容         │
└────────────────────────────────┘
```

- 尺寸: 300x180px
- 位置: 右下角
- 透明度: 90%
- 关闭按钮: 最小化到托盘

### 5.2 配置窗口

- **窗口过滤**: 勾选列表
- **快捷键设置**: 显示/隐藏热键输入
- **数据库管理**: 添加/删除关键词

## 6. 数据表设计

```sql
-- 检索关键词表
CREATE TABLE keywords (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    keyword TEXT NOT NULL,
    description TEXT,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```

## 9. 复制窗口来源功能

- 点击悬浮窗标题旁的复制按钮
- 复制当前来源窗口标题到剪贴板
- 格式: `窗口标题` (如: `Notepad++ - document.txt`)
